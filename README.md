🎮 Fan Game SuperSonic

SuperSonic é um fan game inspirado no universo de Sonic, desenvolvido exclusivamente para fins acadêmicos, sem fins lucrativos.

O projeto foi criado utilizando o Construct 2, uma game engine voltada ao desenvolvimento de jogos 2D baseados em HTML5 e JavaScript, com foco em prototipagem rápida e jogos educacionais.

🎯 Objetivo do Projeto

Aplicar conceitos de:

Desenvolvimento de jogos 2D

Prototipagem rápida

Lógica orientada a eventos

Interação Humano-Máquina (IHM)

Acessibilidade e inclusão digital

O jogo é voltado a pessoas com TEA (Transtorno do Espectro Autista), buscando oferecer uma experiência mais confortável, previsível e acolhedora.

⚙️ Tecnologias Utilizadas

🎮 Construct 2 — Engine de desenvolvimento de jogos 2D

🌐 HTML5 — Plataforma de execução

📜 JavaScript — Lógica do jogo (gerada pelo engine)

🎨 Assets gráficos 2D

O jogo é exportado para ambiente web e pode ser executado diretamente no navegador.

🙋‍♂️Acessibilidade e Conceitos de IHM

O projeto considera princípios de Interação Humano-Máquina (IHM), priorizando usabilidade, conforto cognitivo e previsibilidade da interação.

🖼️ Uso de Imagens e GIFs

As imagens, sprites e GIFs utilizados neste projeto foram obtidos a partir de fontes públicas na internet e são usados exclusivamente para fins educacionais e não comerciais.

Todos os direitos autorais pertencem aos seus respectivos autores e detentores.
Caso algum detentor de direitos solicite a remoção de conteúdo, o material será removido prontamente.
