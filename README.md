🎮 Fan Game SuperSonic

SuperSonic é um fan game inspirado no universo de Sonic, desenvolvido exclusivamente para fins acadêmicos, sem fins lucrativos.
O projeto é voltado a pessoas com TEA (Transtorno do Espectro Autista), priorizando uma experiência mais confortável, com estímulos visuais suaves e foco em acessibilidade.

⚠️ Status do Projeto:
Este jogo encontra-se em versão beta, sendo um protótipo em desenvolvimento.
Atualmente não possui sons ou músicas implementados e requer melhorias técnicas e de usabilidade.
