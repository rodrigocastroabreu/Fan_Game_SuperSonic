# Fan Game SuperSonic

Sonic é um fan game do Sonic criado para um trabalho de faculdade, sem fins lucrativos. O jogo é voltado a pessoas com TEA (Transtorno do Espectro Autista), com sons e visuais suaves e opções de acessibilidade. Busca promover diversão, conforto e inclusão digital.

## Visão Geral

Este projeto é uma homenagem ao universo de Sonic, desenvolvido especialmente para um ambiente educativo e inclusivo. O jogo não tem fins comerciais e foi criado como parte de um trabalho acadêmico.

## Acessibilidade

O jogo inclui:

- Sons e músicas suaves para reduzir estímulos sensoriais.
- Visuais com cores e contrastes mais confortáveis.
- Opções de controle e navegação pensadas em acessibilidade.
